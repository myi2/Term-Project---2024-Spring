Welcome to HomeHarvest: Real Estate Listings Scraper

HomeHarvest is a Python-based tool designed to automate the collection of real estate listings. Specify your desired location, price range, and features to receive tailored property recommendations.

How to Use HomeHarvest

HomeHarvest is a command-line tool that requires Python 3.6 or later. To use HomeHarvest, follow these steps:

  1. Clone the HomeHarvest repository from GitHub.
  2. Install the required Python packages using pip.
  3. Run the HomeHarvest script with your desired search parameters.

HomeHarvest Features

  • Scrape real estate listings based on user-defined criteria.
  • Data output includes property details such as address, price, and features.
  • Flexible search parameters including location, price range, and property type.

Learning Objectives

Through the development of HomeHarvest, our team aimed to:

  • Develop a Python-based web scraper for real estate data.
  • Master data manipulation and formatting for ease of user access.
  • Explore machine learning algorithms to suggest properties based on user preferences.
